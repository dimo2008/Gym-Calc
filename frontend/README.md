# Gym Calc Frontend

Angular frontend for the Gym Calc Dashboard application.

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Development Server**
   ```bash
   npm start
   ```

   The application will be available at `http://localhost:4200`

3. **Build for Production**
   ```bash
   npm run build
   ```

## Features

- **Dashboard**: Overview of members, income, expenses, and net profit
- **Members Management**: Add, edit, delete gym members
- **Payments Management**: Track member payments
- **Expenses Management**: Track gym expenses by category
- **Responsive Design**: Works on desktop and mobile devices

## API Integration

The frontend communicates with the backend API at `http://localhost:3000/api`.

Make sure the backend server is running before using the application.

## Navigation

- **Dashboard**: Home page with overview statistics
- **Members**: Manage gym members
- **Payments**: Track income from members
- **Expenses**: Track gym operational expenses
